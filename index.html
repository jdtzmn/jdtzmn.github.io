About
Jacob is an experienced full-stack web app developer. He started web development in second grade when he was nine years old. In fifth grade, he stayed up for three days straight and started learning JavaScript entirely off of StackOverflow questions. Without any prior programming experience, he had completed his first website within a week. Since then, Jacob has worked on numerous personal and professional projects. Even before his first AP computer science course, Jacob had already built twelve web apps. He has competed on two competitive robotics teams, and is currently taking a second computer science class in functional programming and chip design. He is constantly working on one app or another, and is always learning about new technologies to improve his apps. Jacob is a very enthusiastic and focused individual who thrives working independently or on a team.
Projects
ChRIS Store UI
ChRIS is an open source framework that utilizes cloud technologies to democratize medical analytics application development. The ChRIS Store is a store for all the ChRIS plugins and a place for developers to publish their own.
SpotSync
SpotSync is a web app powered by the Spotify Web Playback SDK that automatically syncs two Spotify accounts to play music at the same time. When the "leader" changes songs, all the listeners' songs change magically to the next song.
Twitch Dashboard
Twitch Dashboard is an electron app that gives a preview of what the viewers see in a window that can be viewed while playing games. It also sends Twitch chat as desktop notifications to the top right corner of the screen.
